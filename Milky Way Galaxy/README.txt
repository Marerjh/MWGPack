HOW-TO INSTALL:
1. Download latest version of the pack from GitHub or GoogleDrive
2. UnZIP it
3. Move "Milky Way Galaxy" folder that you get after unzipping to "Android/data/com.StefMorojna.SpaceflightSimulator/files/Custom Solar Systems"
4. Create new world with "Milky Way Galaxy" as solar system


HOW-TO UPDATE:
1. Download latest version of the pack from GitHub or GoogleDrive
2. UnZIP it
3. Replace old folder of pack with new one
4. Enter your world by taping "Build rocket"

Please note that some of your crafts might disappear or change orbit
